# Projects

**Purpose:** Track active and past projects with progress, milestones, key learnings, and outcomes.

## What Goes Here

- Active projects in progress
- Recently completed projects
- Past projects with learnings
- Project status and milestones
- Key insights and outcomes

## How It Works

1. **Create** - Start with project overview using [template](../.templates/project.md)
2. **Track** - Update progress regularly
3. **Learn** - Capture insights and decisions
4. **Archive** - Move to [Archive](../06-archive/index.md) when complete
