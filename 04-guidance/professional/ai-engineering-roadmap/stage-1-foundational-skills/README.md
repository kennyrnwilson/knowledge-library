## Learning Checklist

### Python Fundamentals
- [ ] Install Python and set up virtual environment
- [ ] Understand variables, data types, and operators
- [ ] Write conditional statements (if/else)
- [ ] Work with loops (for, while)
- [ ] Use lists, dicts, tuples, and sets
- [ ] Write functions with parameters and returns
- [ ] Understand variable scope
- [ ] Write classes and use OOP
- [ ] Handle exceptions with try/except
- [ ] Use debugging tools (print, pdb)
- [ ] Write code following PEP 8
- [ ] Install and use packages with pip/Poetry

### Software Engineering
- [ ] Initialize Git repo and make commits
- [ ] Use GitHub (push, pull, branches)
- [ ] Understand branching workflows
- [ ] Write a proper `.gitignore`
- [ ] Write README documentation
- [ ] Create proper project structure (src-layout)
- [ ] Write unit tests with pytest
- [ ] Achieve 80%+ test coverage on a project
- [ ] Use code formatters (Black)
- [ ] Use type checkers (mypy)
- [ ] Write docstrings for functions
- [ ] Do code reviews

### Data & Databases
- [ ] Read and write CSV files
- [ ] Use Pandas for data manipulation
- [ ] Filter, sort, and group data
- [ ] Create basic data visualizations
- [ ] Understand database fundamentals
- [ ] Write basic SQL queries
- [ ] Use SELECT, WHERE, ORDER BY
- [ ] Write JOIN queries
- [ ] Use sqlite3 in Python
- [ ] Understand ORM concepts (SQLAlchemy)
- [ ] Prevent SQL injection

### Web Development & APIs
- [ ] Understand HTTP methods and status codes
- [ ] Use requests library to make API calls
- [ ] Handle JSON data
- [ ] Handle errors from APIs
- [ ] Build a FastAPI application
- [ ] Create API endpoints
- [ ] Handle request parameters
- [ ] Return proper responses
- [ ] Handle errors in APIs
- [ ] Understand CORS basics
- [ ] Implement API authentication

### Command Line & DevOps
- [ ] Navigate file system with terminal
- [ ] Create and edit files from terminal
- [ ] Use pipes and redirection
- [ ] Write simple shell scripts
- [ ] Understand what localhost is
- [ ] Connect to remote servers with SSH
- [ ] Understand environment variables
- [ ] Deploy application to Heroku/Railway
- [ ] Check logs from deployed application
- [ ] Understand basic monitoring
