# Permanent Notes

**Purpose:** Your personal knowledge library. Well-formed, structured insights organized hierarchically by topic, ready for reference and application.

## What Goes Here

- Core concepts and frameworks
- Structured knowledge about topics
- Integrated ideas from multiple sources
- Actionable principles and practices
- Evergreen knowledge (doesn't become outdated)

## How It Works

1. **Develop** - From [Fleeting Notes](../01-fleeting-notes/README.md) and [Literature Notes](../02-literature-notes/README.md)
2. **Structure** - Using the [note template](../.templates/note.md)
3. **Connect** - Link to related concepts and ideas
4. **Maintain** - Update with new insights over time
